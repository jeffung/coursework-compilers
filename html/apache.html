Apache HTTP Server

Apache HTTP Server (further referred to as Apache) is a server developed and maintained by the Apache Software Foundation. The goal of the project is to provide the world with an open-source HTTP server that works across platforms. Apache was the first server deployed to over a 100 million servers and is the most commonly used server technology. Additionally, Apache has the edge on node.js (further referred to as node) in comparing the number of platforms supported. Both node and Apache have a guiding principle to be friendly to beginners and allow for quick adoption and utilization.

The feature set of Apache includes authentication, SSL and TLS, configuration of logs, caching, gzip,url rewrite and more.

Node however handles all requests in a single thread, single process model. Apache is non-blocking due to its MPM whereas non-blocking in node is an implementation detail.
